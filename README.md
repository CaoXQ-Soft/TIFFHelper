# TIFFHelper

一、软件简介

三味书屋多页TIFF软件是一款专业性质的免费TIFF软件，主要专注于TIFF文件的处理，包括TIFF合并、TIFF拆分、TIFF调整、TIFF统计、TIFF压缩、分辨率检测、颜色检测，以及TIFF查看与编辑功能。
